# Introduction

## System Requirements
<ul>
    <li>Windows 7+ || macOS (lastest version preferred)</li>
    <li>4GB+ RAM</li>
    <li>10GB+ of free disk space</li>
    <li>Chrome browser (alternative browsers are also fine)</li>
    <li>Node.js</li>
</ul>

Ensure that you're able to check the version of node.js and npm<br>

``` 
node -version
npm -version

```
